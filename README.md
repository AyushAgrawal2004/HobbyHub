## üì¶ Installation & Setup

1.  **Clone the Repo**:
    ```bash
    git clone https://github.com/YourUsername/HobbyHub.git
    cd HobbyHub
    ```

2.  **Install Dependencies**:
    ```bash
    # Server
    cd server
    npm install
    
    # Client
    cd ../client
    npm install
    ```

3.  **Environment Variables**:
    Create `.env` files in both `server/` and `client/` (see `.env.example`).

4.  **Launch**:
    ```bash
    # Terminal 1 (Server)
    cd server
    npm start

    # Terminal 2 (Client)
    cd client
    npm run dev
    ```
